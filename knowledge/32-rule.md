# 32 强出线形势分析规则

## 目标

为已确认晋级的球队分析：第一名出线和第二名出线哪一种路径更有利。

## 执行步骤

1. 先通过 `knowledge/32-data.md` 查看 32 强对阵表
2. 再通过 `knowledge/group-rank.md` 查看各组排名
3. 推导第一名出线时可能遇到的对手
4. 推导第二名出线时可能遇到的对手
5. 对两种路径进行比较，给出更有利的判断

## 推导逻辑说明

### 第一名出线

- 根据该球队组别确定理论对阵区间
- 例如：A组第一，可能对应某些组的第三名
- 结合当前小组排名，推测这些组最可能以第三名出线的球队

### 第二名出线

- 根据该球队组别确定理论对阵区间
- 例如：A组第二，可能直接面对 B 组第二
- 如果相关小组比赛已结束，则直接读取结果
- 如果未结束，则结合当前排名推测最可能的第二名球队

## 输出格式

```md
## 第一名可能遇到的对手：
- C组 xxx
- E组 xxx
- F组 xxx
- H组 xxx
- I组 xxx

## 第二名可能遇到的对手：
- B组 xxx
